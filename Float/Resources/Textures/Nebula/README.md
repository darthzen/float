# Nebula — L2 nebula reference & sprite source (§7 L2, §10)

For the particle-fallback nebula (`ParticleNebula` in `DepthLayers.swift`).

**What goes here:**
- Nebula stills used as **color-palette reference** (sample real RGBs to replace
  the hand-picked palette) and as source to cut into **soft sprite cards** /
  filament textures for the additive billboards.
- Processed, reasonably-sized textures only — full-res originals stay in `_raw/`.

**Note:** the soft radial-alpha *puff masks* for the sprites are generated
procedurally in code, so they don't need to live here.

- ESO — CC BY 4.0 (credit required).
- NASA image galleries — generally public domain (avoid the NASA logo / implying
  endorsement).
